## 📌 Overview

This project predicts **building energy consumption (kWh)** using Machine Learning techniques. It takes real-world parameters like building size, environmental conditions, and time-based features to estimate energy usage.

The system is deployed as an **interactive web application using Streamlit Cloud**, allowing real-time predictions.

---

## 🎯 Objective

* Predict building energy usage accurately
* Demonstrate an end-to-end ML workflow
* Provide real-time prediction using a web interface
* Support energy optimization and smart systems

---

## 🧠 Machine Learning Pipeline

Data Collection → Data Cleaning → Feature Engineering → Scaling → PCA → Model Training → Prediction → Deployment

---

## ⚙️ Technologies Used

* ☁️ Streamlit Cloud

---

## 🔍 Features Used

* Square Feet (Building Size)
* Air Temperature
* Cloud Coverage
* Hour
* Day
* Month

---

## 🤖 Model Details

* Algorithm: **Linear Regression**
* Preprocessing:

  * StandardScaler
  * PCA (Dimensionality Reduction)

---

## 📊 How It Works

1. User enters building and environmental details
2. Input data is scaled using StandardScaler
3. PCA reduces feature complexity
4. Trained model predicts energy consumption
5. Output is displayed in kWh
